# Pynq_es: tutorial para el uso del entorno de productividad Pynq (Pynq-Z2)

## Hardware requerido

* La placa Zynq-Z2
* Ordenador con un navegador compatible (Chrome, Firefox)
* Cable de Ethernet
* Cable de Micro USB
* Memoria Micro SD con la distribución de Linux precargada
